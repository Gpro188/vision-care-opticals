This is a preview of the new offer slider feature for your Vision Care website!

The slider at the top shows promotional offers and only appears when you activate it.

🔄

Smooth infinite sliding animation

✖️

Closeable

How to Use:

  1. Click "Activate Demo Offers" to see it in action
  2. The slider will appear at the top with special offers
  3. Click the × button to close it
  4. To add real offers, use the browser console code below

Browser Console Code:

// Add this to your browser console (F12) const adminData = JSON.parse(localStorage.getItem('visionCareAdminData')); adminData.offers = { active: true, startDate: "2025-01-01", endDate: "2025-12-31", items: [ { icon: "🎉", text: "50% OFF on All Frames!" }, { icon: "🔥", text: "Buy 1 Get 1 Free - Contact Lenses" }, { icon: "⭐", text: "Free Eye Checkup with Purchase" } ] }; localStorage.setItem('visionCareAdminData', JSON.stringify(adminData)); console.log('✅ Offers activated!'); location.reload();
