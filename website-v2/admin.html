📸 Offer Items

Add promotional offers with images. Each offer will slide across the screen.

📋 Offer Design Guidelines:

  • Image Size: 600×100px (landscape/horizontal - compact banner)
  • Design: Add your offer text INSIDE the image using Canva/Photoshop
  • Examples: "EID SALE 50% OFF", "NEW YEAR BOGO", etc.
  • Format: JPG, PNG, or WebP
  • File Size: Max 100KB for fast loading
  • Best Practice: Create beautiful ads with text already designed in them!
  • Why 600×100? Compact horizontal banner that scrolls beautifully without taking too much space!
